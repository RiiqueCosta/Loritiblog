PACOTE COMPLETO - LORI-TI

Arquivos principais:
- index.html -> página inicial
- sobre.html -> quem somos
- servicos.html -> serviços
- blog.html -> listagem do blog
- contato.html -> formulário que abre o WhatsApp com mensagem pronta
- landing-page.html -> página focada em anúncio e conversão
- blog/posts/ -> 20 artigos SEO
- assets/css/style.css -> estilo do site
- assets/js/main.js -> botões do WhatsApp, formulário e triagem automática
- assets/js/posts.js -> lista dos posts do blog
- robots.txt e sitemap.xml -> base para SEO

COMO PUBLICAR NO NETLIFY:
1. Extraia a pasta.
2. Entre no Netlify.
3. Arraste a pasta inteira do projeto para publicar.
4. Se usar domínio próprio, aponte o domínio no painel do Netlify.
